# 预约系统

这是一个基于Next.js开发的预约系统，包含时间选择和用户信息收集功能。

## 主要功能

1. **时间选择功能**
   - 智能日期选择器
   - 精细化时间区块
   - 动态加载逻辑

2. **用户信息收集表单**
   - 必填字段（姓名、手机号）
   - 选填字段（备用联系方式、咨询备注）
   - 智能验证功能

## 技术栈

- Next.js
- React
- TypeScript
- Tailwind CSS

## 本地运行

```bash
npm run dev
# 或
yarn dev
# 或
pnpm dev
# 或
bun dev
```

访问 [http://localhost:3000](http://localhost:3000) 查看应用。
